# 日本三大地標景點介紹網站

介紹大阪城天守閣、雷門、東京晴空塔三個日本地標的靜態網站。

## 檔案結構

```
├── index.html        # 首頁（景點卡片、學習心得、班級學號）
├── osaka.html        # 大阪城天守閣 — 歷史、地理、人文、交通、美食
├── kaminarimon.html  # 雷門 — 完整介紹
├── skytree.html      # 東京晴空塔 — 完整介紹
├── 1.png             # 大阪城天守閣圖片
├── 2.png             # 雷門圖片
├── 3.png             # 晴空塔圖片
└── README.md
```

## 使用方式

直接用瀏覽器開啟 `index.html` 即可瀏覽。
